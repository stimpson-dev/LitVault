  <name>Task 1: Implement PDF parser with OCR detection</name>
  <files>backend/app/ingest/parsers/__init__.py, backend/app/ingest/parsers/pdf_parser.py</files>
  <action>
  Create `backend/app/ingest/parsers/__init__.py` with:

  ```python
  from dataclasses import dataclass, field

  @dataclass
  class ParseResult:
      text: str = ""
      page_count: int = 0
      has_text: bool = False
      ocr_pages: list[int] = field(default_factory=list)
      error: str | None = None
  ```

  Create `backend/app/ingest/parsers/pdf_parser.py` with:

  1. `_is_scanned_page(page: fitz.Page) -> bool` —
     - Get text with `page.get_text()`. If len(text.strip()) < 50 chars, likely scanned.
     - Also check fonts: if all fonts contain "GlyphlessFont" → already OCR'd, don't re-OCR.
     - Return True only if text is sparse AND no GlyphlessFont (truly scanned, needs OCR).

  2. `_normalize_german_text(text: str) -> str` —
     - NFC normalize: `unicodedata.normalize("NFC", text)`
     - Rejoin hyphenated line breaks: `re.sub(r'(\w)-\n(\w)', r'\1\2', text)` (e.g., "Zahn-\nrad" → "Zahnrad")
     - Return cleaned text

  3. `_ocr_page(page: fitz.Page, languages: str = "deu+eng") -> str` —
     - Render page to pixmap (300 DPI): `page.get_pixmap(dpi=300)`
     - Convert to PIL Image
     - Run `pytesseract.image_to_string(img, lang=languages)`
     - Return OCR text

  4. `def parse_pdf(path: Path) -> ParseResult` —
     - Try opening with fitz.open(). If encrypted: try empty password. If still encrypted → return ParseResult(error="encrypted").
     - Try pymupdf4llm.to_markdown(str(path)) for full document. This is the primary extraction.
     - If result text is very short (< 100 chars for multi-page doc), fall back to per-page processing:
       - For each page: check _is_scanned_page
       - If scanned: _ocr_page, collect page number in ocr_pages
       - If text: use page.get_text()
       - Concatenate all page texts
     - Apply _normalize_german_text to final text
     - Return ParseResult with text, page_count, has_text=(len(text.strip()) > 0), ocr_pages
     - Wrap entire function in try/except: on any exception → ParseResult(error=str(e))

  5. `async def parse_pdf_async(path: Path) -> ParseResult` —
     - `asyncio.wait_for(asyncio.to_thread(parse_pdf, path), timeout=30.0)`
     - On TimeoutError → ParseResult(error="timeout")

  Use `logger = logging.getLogger("litvault.parser.pdf")`

  <action>
  Create `backend/app/ingest/parsers/office_parser.py` with:

  1. `def parse_docx(path: Path) -> ParseResult` —
     - Use python-docx: `Document(str(path))`
     - Extract text from all paragraphs: `"\n".join(p.text for p in doc.paragraphs)`
     - Also extract text from tables: iterate doc.tables → rows → cells → text
     - NFC normalize the result
     - Return ParseResult with text, page_count=0 (DOCX has no page concept), has_text
     - try/except → ParseResult(error=str(e))

  2. `def parse_pptx(path: Path) -> ParseResult` —
     - Use python-pptx: `Presentation(str(path))`
     - Extract text from all slides → shapes → text_frame → paragraphs
     - page_count = len(prs.slides)
     - NFC normalize the result
     - Return ParseResult with text, page_count, has_text
     - try/except → ParseResult(error=str(e))

  3. `async def parse_docx_async(path: Path) -> ParseResult` / `parse_pptx_async` —
     - Same pattern: asyncio.to_thread + wait_for 30s

  Update `backend/app/ingest/parsers/__init__.py` — add dispatcher:

  ```python
  from .pdf_parser import parse_pdf_async
  from .office_parser import parse_docx_async, parse_pptx_async

  PARSERS: dict[str, Callable] = {
      "pdf": parse_pdf_async,
      "docx": parse_docx_async,
      "pptx": parse_pptx_async,
  }

  async def parse_document(path: Path, file_type: str) -> ParseResult:
      parser = PARSERS.get(file_type)
      if not parser:
          return ParseResult(error=f"Unsupported type: {file_type}")
      return await parser(path)
  ```
