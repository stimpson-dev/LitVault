  <action>
  Create `backend/app/classification/__init__.py` (empty).

  Create `backend/app/classification/ollama_client.py` with:

  1. `class OllamaClient`:
     - `__init__(self, base_url: str = "http://localhost:11434", model: str = "qwen3:8b")`:
       - Store base_url, model
       - Create `httpx.AsyncClient(base_url=base_url, timeout=120.0)` — 120s timeout for LLM calls

     - `async def generate(self, prompt: str, json_schema: dict | None = None) -> dict`:
       - POST to `/api/generate` with body:
         ```json
         {
           "model": self.model,
           "prompt": prompt,
           "stream": false,
           "options": {
             "temperature": 0,
             "num_predict": 2048
           },
           "format": json_schema if json_schema else "json",
           "keep_alive": -1
         }
         ```
       - NOTE on Ollama structured output: The `format` field accepts a JSON schema dict directly.
         Ollama will constrain the output to match this schema. Pass the Pydantic model's
         `.model_json_schema()` output as the schema.
       - Parse response JSON, extract `response` field
       - Parse the `response` string as JSON (it's a JSON string inside the response)
       - Return the parsed dict
       - On httpx errors: log and raise
       - On JSON parse errors: log and raise ValueError

     - `async def check_health(self) -> bool`:
       - GET `/api/tags` — if 200, return True
       - On error: return False

     - `async def close(self)`:
       - await self._client.aclose()

  Use `import httpx, json, logging`
  Use `logger = logging.getLogger("litvault.ollama")`

<task type="auto">
  <name>Task 2: Classification schemas, prompt, and service</name>
  <files>backend/app/classification/schemas.py, backend/app/classification/service.py</files>
  <action>
  Create `backend/app/classification/schemas.py` with:

  1. Constants:
     ```python
     DOC_TYPES = [
         "dissertation", "paper", "norm", "bericht",
         "präsentation", "artikel", "interne_notiz"
     ]

     CATEGORIES = [
         "Verzahnungsgrundlagen",
         "Kegelrad / Hypoid / Stirnrad",
         "Tragbild / Kontakt / NVH",
         "FEM / Spannungen / Lebensdauer",
         "Werkstoffe / Wärmebehandlung",
         "Fertigung / Schleifen / Honen / Härten",
         "Prüfstand / Versuch / Schadensanalyse",
         "Normen / ISO / DIN / AGMA / FVA",
         "Anwendungen / Differential / E-Achse / Nutzfahrzeug",
         "Interne Berichte / Projektdokumente",
     ]
     ```

  2. Pydantic model `ClassificationResult(BaseModel)`:
     - doc_type: str  # one of DOC_TYPES
     - categories: list[str]  # from CATEGORIES vocabulary
     - tags: list[str]  # free-form keywords, 3-8 tags
     - summary: str  # 2-3 sentence summary in document language
     - title: str
     - authors: list[str]
     - year: int | None = None
     - source: str | None = None  # journal, university, publisher
     - confidence: float  # 0.0-1.0 overall confidence

  3. `CLASSIFICATION_PROMPT` template string:
     ```python
     CLASSIFICATION_PROMPT = """Analyze this engineering/scientific document excerpt and extract structured metadata.

     DOCUMENT TYPES (pick exactly one):
     {doc_types}

     CATEGORIES (pick 1-3 from this list only):
     {categories}

     RULES:
     - Extract title, authors, year, and source/journal if present
     - Generate 3-8 keyword tags relevant to the content
     - Write a 2-3 sentence summary in the document's language
     - Set confidence between 0.0 and 1.0 based on how clearly you can identify the metadata
     - If information is unclear or missing, use null/empty and lower confidence

     DOCUMENT TEXT:
     {text}"""
     ```

  4. `def build_prompt(text: str) -> str` — fills in the template with DOC_TYPES and CATEGORIES lists.

  Create `backend/app/classification/service.py` with:

  1. `CONFIDENCE_AUTO = 0.85`
     `CONFIDENCE_REVIEW = 0.55`
     `MIN_WORDS = 30`
     `MAX_CHARS = 2000`

  2. `def detect_language(text: str) -> str | None`:
     - Use `from langdetect import detect, LangDetectException`
     - try: return detect(text)
     - except LangDetectException: return None

  3. `def truncate_text(text: str, max_chars: int = MAX_CHARS) -> str`:
     - If len(text) <= max_chars: return text
     - Truncate at max_chars, find last space before that point to avoid cutting words
     - Append "..." to indicate truncation
     - Return truncated text

  4. `def classify_by_filename(filename: str) -> dict | None`:
     - Simple rule-based fallback for short/garbled docs
     - Check filename patterns:
       - Contains "DIN" or "ISO" or "AGMA" → doc_type="norm", category="Normen / ISO / DIN / AGMA / FVA"
       - Contains "Bericht" or "Report" → doc_type="bericht"
       - Contains "Präsentation" or "Presentation" → doc_type="präsentation"
       - Contains "FVA" → category="Normen / ISO / DIN / AGMA / FVA"
     - Return dict with partial metadata or None if no rules match
     - This is a FALLBACK only — used when text is too short for LLM

  5. `class ClassificationService`:
     - `__init__(self, ollama: OllamaClient)`

     - `async def classify(self, text: str, filename: str = "") -> ClassificationResult`:
       a. Count words. If < MIN_WORDS:
          - Try filename-based rules
          - If rules match: return ClassificationResult with low confidence (0.3) and source info
          - If no rules: return ClassificationResult with confidence=0.0 (unclassified)
       b. Truncate text to MAX_CHARS
       c. Build prompt via build_prompt(truncated)
       d. Get JSON schema from ClassificationResult.model_json_schema()
       e. Call self.ollama.generate(prompt, json_schema=schema)
       f. Parse response into ClassificationResult via ClassificationResult.model_validate(response)
       g. Clamp confidence to 0.0-1.0
       h. Return result
       - On any exception: log error, return ClassificationResult with confidence=0.0, error info in summary

     - `def get_confidence_tier(self, confidence: float) -> str`:
       - If >= CONFIDENCE_AUTO: return "auto"
       - If >= CONFIDENCE_REVIEW: return "needs-review"
       - Else: return "unclassified"

     - `async def classify_document(self, text: str, filename: str = "") -> tuple[ClassificationResult, str]`:
       - Calls classify(), then get_confidence_tier()
       - Returns (result, tier)

  Use `logger = logging.getLogger("litvault.classification")`
