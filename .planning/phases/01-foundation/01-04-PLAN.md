  **backend/app/main.py:**
  1. Import FastAPI, CORSMiddleware, StaticFiles
  2. Create `lifespan` async context manager:
     - On startup: create all tables (run_sync create_all), log "LitVault started"
     - On shutdown: log "LitVault stopped"
     - (Worker tasks will be added in Phase 3)
  3. Create FastAPI app with:
     - title="LitVault"
     - lifespan=lifespan
  4. Add CORS middleware:
     - allow_origins: ["http://localhost:5173"] (Vite dev server)
     - allow_methods: ["*"]
     - allow_headers: ["*"]
  5. Create health endpoint:
     ```python
     @app.get("/api/health")
     async def health():
         return {"status": "ok", "service": "litvault"}
     ```
  6. Add conditional static files mount (only if frontend/dist exists):
     ```python
     frontend_dist = Path(__file__).parent.parent.parent / "frontend" / "dist"
     if frontend_dist.exists():
         app.mount("/", StaticFiles(directory=frontend_dist, html=True), name="static")
     ```
  7. Create start scripts:
     - backend/start.bat: `cd /d %~dp0 && uv run uvicorn app.main:app --reload --port 8000`
     - frontend/start.bat: `cd /d %~dp0 && npm run dev`
  </action>
