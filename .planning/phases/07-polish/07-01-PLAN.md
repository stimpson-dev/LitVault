  <name>Task 1: Favorites + Tags + Document update APIs</name>
  <files>backend/app/documents/router.py</files>
  <action>
  Extend `backend/app/documents/router.py` with:

  **Favorites:**
  - `POST /api/documents/{doc_id}/favorite` — Toggle favorite (add if not exists, remove if exists). Returns { favorited: bool }.
  - `GET /api/favorites` — List all favorited documents (join Favorite → Document), ordered by added_at desc. Returns list of document dicts.

  **Tags on a document:**
  - `GET /api/documents/{doc_id}/tags` — List tags for a document. Returns list of { id, name, source }.
  - `POST /api/documents/{doc_id}/tags` — Add a tag by name (create Tag if not exists, create DocumentTag with source="user"). Body: { name: string }. Returns { id, name }.
  - `DELETE /api/documents/{doc_id}/tags/{tag_id}` — Remove tag from document.

  **Document metadata update:**
  - `PATCH /api/documents/{doc_id}` — Update editable fields: title, authors, year, doc_type, source, language, summary. Body: partial dict. Sets classification_source to "user" when any AI field is overridden. Returns updated document dict.

  **CSV export:**
  - `GET /api/search/export` — Same params as /api/search but returns CSV. Uses StreamingResponse with text/csv. Columns: title, authors, year, doc_type, source, language, file_path, file_type, file_size. Limit to 1000 rows max.

  Use existing models: Document, Tag, DocumentTag, Favorite.
  Import select, delete from sqlalchemy as needed.
  </action>

  </verify>
